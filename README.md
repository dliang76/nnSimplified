# Project
nnSimplified

# Description
The package simplifies creation of complex neural network structure using torch framework
# Prerequisite
Need graphvis library to work on Mac
How to install graphvis:
Homebrew:
```
brew install graphviz
```
MacPorts:
```
sudo port install graphviz
```

# Installation
```
pip install git+https://github.com/dliang76/nnSimplified.git
```
