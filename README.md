# Project
nnSimplified

# Description
The package simplifies creation of complex neural network structure using torch framework
# Prerequisite
Need graphvis library to work on Mac
How to install graphvis:
Homebrew:
```
brew install graphviz
```
MacPorts:
```
sudo port install graphviz
```

# Installation
1. Clone the repo
```
git clone https://github.com/dliang76/nnSimplified.git
```

2. Install the package
```
pip install nnSimplified/.
```

# Examples
There are some simple examples on how to construct certain neural net structures in nnSimplified/notebooks/some_examples.ipynb
